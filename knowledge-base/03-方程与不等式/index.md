# 方程与不等式 — 知识点内容

## MATH-03-001 一元一次方程

标准形式 ax+b=0（a≠0），解为 x=-b/a。它是方程世界的基础。

**核心概念：**
- 方程：含有未知数的等式
- 解（根）：使方程成立的未知数的值
- 解方程：求方程的解的过程
- 同解原理：方程两边同时加减/乘除同一个非零数，解不变

**解法五步：** 去分母 → 去括号 → 移项 → 合并同类项 → 系数化为 1

## MATH-03-002 一元二次方程

标准形式 ax²+bx+c=0（a≠0），核心是判别式 Δ=b²-4ac。

**核心概念：**
- 四种解法：直接开平方、配方法、公式法、因式分解法
- 判别式 Δ=b²-4ac：>0 两个不等实根 / =0 重根 / <0 无实根
- 韦达定理：x₁+x₂=-b/a，x₁·x₂=c/a

## MATH-03-003 分式方程

分母中含有未知数的方程。

**核心要点：**
- 去分母 → 解整式方程 → 验根（必须！）
- 增根：去分母过程产生的使分母为 0 的"假解"
- 分式方程可能有解、无解、有增根三种情况

## MATH-03-004 方程组

多个方程联立求解，核心思想是消元。

**核心概念：**
- 代入消元法：将一式写成 y=... 代入另一式
- 加减消元法：两式相加减消去一个未知数
- 二元一次方程组解有三种情况：相交（唯一解）/ 平行（无解）/ 重合（无数解）

## MATH-03-005 一元一次不等式

用不等号连接的大小关系。

**黄金法则：** 两边同乘负数，不等号方向反转！

## MATH-03-006 一元二次不等式

二次函数与不等式的结合，"大于取两边，小于取中间"。

**解法三步：**
1. 解对应二次方程求根
2. 根据二次函数图像开口方向
3. 确定解集范围
