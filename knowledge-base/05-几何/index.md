# 几何 — 知识点内容

## MATH-05-001 点、线、面、体
几何四大基本元素，点动成线、线动成面、面动成体。
两点确定一条直线，两点之间线段最短。

## MATH-05-002 三角形
最稳定的多边形。内角和 180°。
分类：按角分（锐角/直角/钝角），按边分（等腰/等边/不等边）。
重要线段：高、中线、角平分线、中位线。

## MATH-05-003 全等三角形
形状大小完全重合。判定：SSS/SAS/ASA/AAS/HL。
对应边相等，对应角相等。

## MATH-05-004 相似三角形
形状相同、大小成比例。判定：AA/SAS/SSS。
周长比 = 相似比 k，面积比 = k²。

## MATH-05-005 勾股定理
a² + b² = c²（直角三角形）。
经典勾股数：(3,4,5), (5,12,13), (8,15,17)。

## MATH-05-006 四边形
平行四边形、矩形、菱形、正方形、梯形。
平行四边形的判定：一组对边平行且相等。

## MATH-05-007 圆
到定点距离等于定长的点的集合。
周长 C=2πr，面积 S=πr²。
垂径定理、圆周角定理、切线性质。

## MATH-05-008 立体几何初步
空间线面关系：平行、相交、异面。
常见几何体：棱柱、棱锥、圆柱、圆锥、球。
柱体 V=Sh，锥体 V=Sh/3，球体 V=4πr³/3。

## MATH-05-009 解析几何初步
用坐标研究几何 — 笛卡尔的创举。
直线方程、圆的方程、距离公式。
