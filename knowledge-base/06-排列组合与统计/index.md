# 排列组合与统计 — 知识点内容

## MATH-06-001 计数原理
分类加法（或的关系）与分步乘法（且的关系）— 排列组合的两大基石。

## MATH-06-002 排列
从 n 个取 m 个排成一列，顺序重要。
A(n,m) = n!/(n-m)!。
全排列 A(n,n)=n!。

## MATH-06-003 组合
从 n 个取 m 个组成一组，顺序不重要。
C(n,m) = n!/[m!(n-m)!]。
对称性：C(n,m)=C(n,n-m)。

## MATH-06-004 二项式定理
(a+b)ⁿ = Σ C(n,k) a^(n-k) b^k。
系数就是杨辉三角/帕斯卡三角的每一行。
系数之和 = 2ⁿ。

## MATH-06-005 概率
P(A) = 事件A结果数/全部可能结果数。
0≤P≤1，对立事件 P(Ā)=1-P(A)。
互斥事件 P(A∪B)=P(A)+P(B)。
独立事件 P(A∩B)=P(A)·P(B)。

## MATH-06-006 统计
描述统计：均值、中位数、众数、方差、标准差。
抽样：简单随机、系统抽样、分层抽样。
数据可视化：条形图、直方图、散点图、箱线图。
