# 数列与导数 — 知识点内容

## MATH-07-001 数列的概念
按一定次序排列的一列数 {aₙ}。
通项公式 aₙ=f(n)，递推公式 a_{n+1}=f(aₙ)。

## MATH-07-002 等差数列
每项与前一项的差为常数 d。
aₙ = a₁ + (n-1)d。
求和 Sₙ = n(a₁+aₙ)/2 = na₁ + n(n-1)d/2。
高斯求和：1+2+...+100=5050。

## MATH-07-003 等比数列
每项与前一项的比为常数 q。
aₙ = a₁·q^(n-1)。
求和 Sₙ = a₁(1-qⁿ)/(1-q)（q≠1）。
与指数函数关系紧密。

## MATH-07-004 数列求和
裂项相消：1/[n(n+1)] = 1/n - 1/(n+1)。
错位相减：等差×等比型。
分组求和：拆成两个数列分别求和。

## MATH-07-005 导数概念
导数 f'(x₀) = lim[f(x₀+Δx)-f(x₀)]/Δx (Δx→0)。
几何意义：切线斜率。
导数>0 增函数，导数<0 减函数。

## MATH-07-006 导数计算
基本公式：(xⁿ)'=nxⁿ⁻¹, (eˣ)'=eˣ, (ln x)'=1/x, (sin x)'=cos x。
四则运算法则：和差、乘积、商。
链式法则：复合函数求导。

## MATH-07-007 导数应用
单调性判定：f'(x)>0 递增，f'(x)<0 递减。
极值：f'(x₀)=0 且变号→极值点。
最值：极值+端点值中取最大/最小。
实际应用：优化问题、物理速度、切线方程。
