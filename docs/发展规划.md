## 五、内容深化计划

每个知识点的 contentJson 分阶段增加以下板块：

| 板块 | 内容 | 当前覆盖 |
|------|------|:--------:|
| **strategy** | 解题思维/策略 | ~60/400+ 节点 |
| **common-mistakes** | 易错点 | ~60/400+ 节点 |
| **derivation** | 推导证明 | 少量 |
| **analogy** | 类比理解 | 少量 |
| **延伸拓展** | 变式、推广 | 规划中 |

目标：每个知识点确保**策略 + 易错 + 推导/类比**三件套，80% 以上达标。
